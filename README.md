# Splunk-Sysmon-Threat-Monitoring-Dashboard
Splunk dashboard using Sysmon logs for endpoint monitoring and threat detection
This project demonstrates Windows threat monitoring using Sysmon logs and Splunk.

## Tools Used
- Splunk Enterprise
- Sysmon
- Windows Event Logs

## Detection
- Suspicious process monitoring
- PowerShell activity
- Command line monitoring

## SPL Query
source="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
